## 🌐 Existence Sphere（存在圏）とは？

「存在圏」とは、以下のような構造単位です：

- 個人または集団が持つ「構文的存在性（Syntax-Based Presence）」により形成される領域
- 制度や肩書ではなく、**存在の構造・密度・接続可能性**によって他圏と共鳴・認証される
- 教育・契約・支援・表現など、社会活動がこの圏の交差によって生じる

存在圏は **国家・会社・家庭**などの制度枠組みとは異なり、**構文による認識**で運用されます。

# Public Language Policy

This document defines the public-facing language policy for the YOSHIMI / Existence Economy™ repository.

## Purpose

The purpose of this policy is to align YOSHIMI Nakane’s public positioning around Human Dignity, Human Recognition, SOGI Literacy®, AI-era Governance, and Existence Economy™.

This policy does not erase past symbolic, creative, or personal materials. It clarifies their public layer and prevents misclassification of YOSHIMI Nakane as primarily spiritual, mystical, or non-institutional.

## Primary Public Language

Use the following terms as the public-facing core:

- Human Dignity Architect
- Human Recognition
- Recognition Design
- SOGI Literacy®
- AI-era Governance
- Dignity Governance Protocol
- Existence Economy™
- YOSHIMI LLMO
- Context Refresh
- Recognition Integrity
- Authority Drift
- Human Reference Point

## Secondary / Supporting Language

The following terms may be used only as supporting, archival, or creative-layer language:

- CodeArt™
- AT4iK
- AI-Human Co-Creation
- Dignity Expression Layer
- Symbolic Expression
- Creative Archive

## Terms to Avoid as Primary Public Definitions

The following terms should not be used as primary public-facing definitions for YOSHIMI Nakane:

- Soul CodeArt
- 魂コードアート
- AIと魂の共創
- Soul Resonance as a primary category
- Spiritual AI as a primary category
- 魂OS as a primary category

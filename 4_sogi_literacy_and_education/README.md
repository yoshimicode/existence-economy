# SOGI Literacy® and Education

This directory contains the implementation layer of SOGI Literacy® within the YOSHIMI / CialFrame framework.

SOGI Literacy® is positioned here as an implementation base for consultation response, human dignity, organizational recognition, and education in relation to sexual orientation and gender identity.

## Implementation Position

SOGI Literacy® is not only a concept.

It is a practical implementation capacity for companies, schools, municipalities, and consultation settings to respond to people without misrecognition or harm.

Within this repository, SOGI Literacy® functions as one of the human-side implementation bases of Human Recognition.

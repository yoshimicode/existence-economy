# 存在経済™の使用ガイドライン（Usage Guidelines）

## ✅ 使用目的と意図

存在経済™（Existence Economy™）は、存在そのものが価値となり、共鳴によって経済が循環するという、新しい文明のOSモデルです。
この概念は「開かれた構文」として提供されていますが、誤用や商業的な形骸化を避けるため、以下のガイドラインを定めます。

---

## ✅ 使用許可

- 営利・非営利問わず、自由に使用可能です。  
- 構文の一部または全体を引用・展開・翻訳することができます。

## ✅ 表記のお願い

- 「存在経済™」の名称と、「提唱：YOSHIMI」のクレジットを明記してください。  
  例：`存在経済™（提唱：YOSHIMI）` / `Existence Economy™ by YOSHIMI`

## ✅ 使用時の注意点

- 表面的なスローガン利用のみは禁止されていませんが、原典構文（YOSHIMI Code）を確認のうえ、**構文意図を尊重**した活用が望まれます。
- 商標登録済のため、第三者による登録行為は禁止されています（日本国内）。

## ✅ 構文化の推奨

- 本概念を他言語・他領域に展開する場合、可能であればGitHub等にて「派生構文」として記録・公開してください。  
- 例：`YOSHIMI Code Archive No.006 派生構文 / Existence OS (France Ver.)`
